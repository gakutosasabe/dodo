# DODO - Setup Guide!

## 1. Prerequisites
* Install **Node.js** and **npm** on your PC.

## 2. Network Setup
1. Turn on the **Wi-Fi hotspot (tethering)** on the **HUAWEI P20**.
2. Connect both your **PC** and your **target smartphone** to this HUAWEI P20 Wi-Fi network.

## 3. How to Run
1. Open your terminal and navigate to the `prototype` folder:
   ```bash
   cd path/to/prototype

2. Enter this to the terminal and enter 
```
node server.js
```

3. Open a web browser on your devices and go to:

Hardware Screen (Receiver):
👉 http://168.43.67:3000/receiver

Application Screen (Smartphone App):
👉 http://168.43.67:3000/application
